My Projects

Photography Spot Finder

A web app for photographers to discover, share, and rate the best photo spots in New York City. Users can browse locations, submit new spots with descriptions and images, leave ratings and comments, and save favorites to their profiles. A built-in map allows searching by location, and a monthly photo contest lets users submit and vote on the best shots. The platform also includes user verification for credibility and an admin panel to moderate reported content.

Node.js Express MongoDB Handlebars HTML5 CSS3 JavaScript Mapbox API (maps) Cloudinary (image storage) Jest (testing) GitHub Actions (CI/CD)
